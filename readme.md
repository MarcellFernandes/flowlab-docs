# FlowLab — Technical Writing Portfolio Project

This repository demonstrates structured technical documentation for a fictional laboratory workflow platform called FlowLab.

The project showcases:

* System overview documentation
* Operational workflow modeling
* Quickstart integration guide
* Use case lifecycle example
* Architecture diagrams

Built with MkDocs to simulate real-world documentation delivery.

## Run locally

```bash
python -m mkdocs serve
```

Then open:

```
http://127.0.0.1:8000/
```

## Purpose

This project exists purely for portfolio demonstration — showing how operational processes can be translated into clear, professional technical documentation.
